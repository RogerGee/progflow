About ProgFlow

ProgFlow is a simple, visual programming language modeled after program flow diagramming conventions. The language provides most standard programming constructs such as decision, iteration, I/O and procedure structures. The simulator provides a command-line interface for interactively entering input and viewing output.

ProgFlow was developed by Roger Gee at Abilene Christian University. If you would like to file a bug report, please email the original author.
